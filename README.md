# コンソールで単語とかを暗記するやつ

コンソール（ターミナル）上で動作するシンプルな単語暗記ツールです。  
単語と意味の対応表（table）を作成し、4択形式で暗記を行うことができます。

---

# How to use

使い方は簡単です。緑色のボタン **`<> Code ▼`** をクリックして出てきたメニューの **`Download ZIP`** をクリックします。

もしくは以下のリンクから直接ダウンロードすることもできます。

https://github.com/MonakaLabo/PresentForYou/archive/refs/heads/main.zip

```
PresentForYou
|
+-history
+-tables
|   |
|   +-sample1
|   +-sample2 ...
+-main.py
+-table_editor.py
+-README.md ...
```

ダウンロードしたZIPファイルを展開し、フォルダをエクスプローラーで開きます。

```
main.py
```

を実行すると暗記を開始できます。

---

# main.pyについて

このシステムの主要プログラムです。  
単語の出題と解答処理を行います。

入力はすべて **半角数字およびカンマ**で行います。

---

# 出題内容選択について

プログラムを実行すると、出題設定を行うプロトコルが開始します。  
入力内容は以下の順序で求められます。

> [!TIP]
> 選択番号が **0から始まる項目**は、何も入力せず Enter を押すと **0番が選択**されます。

1. モード選択  
```
0: 暗記
1: 履歴
```

2. 出題table選択（複数選択可能）

3. 出題順選択  
```
0: ランダム
1: 正順
2: 逆順
```

4. 出題方向選択  
```
0: 表向き
1: 裏向き
```

5. 出題数入力

ただし**5. 出題数入力**は、何も入力せず Enter を押すと**すべて出題**します。

---

## table選択について

複数のtableを選択することができます。

例

```
1,2
```

のように入力すると、両方のtableが出題対象になります。

---

## 出題方向について

「表向き」は**単語->意味**、「裏向き」は**意味->単語**です。

単語帳を裏返して確認するイメージです。

---

# 出題形式について

表示された単語について、正しい選択肢を **1～4の中から1つ選択**します。

正誤判定は即時に行われ、設定した出題数に達するまで連続して問題が出題されます。

画面は次のようなイメージです。

```
----------------
問題 3/20
前回 5.31s
累計 15.48s
正答率 66.7%
単語番号 14
----------------

apple

1: りんご
2: ぶどう
3: みかん
4: いちご

>
```

---

# table_editor.pyについて

tableファイル(単語と意味の対応表)を作成・編集するための支援ツールです。

直接実行で起動できます。

このツールを使用して

- 新規単語帳の作成
- 既存単語帳の編集(内容の追加)

を行うことができます。

---

# tableファイルの構造

tableファイルは次のような構造になっています。

```
tags,tags
description
label1,label2
True,11

apple   りんご
orange  オレンジ
banana  バナナ
```

意味は次の通りです。

|行|内容|
|---|---|
|1|タグ|
|2|説明|
|3|ラベル|
|4|単語番号設定|
|5以降|単語と意味|

---

## 単語番号設定

4行目の設定

```
True,11
```

の場合

```
11
12
13
...
```

のように単語番号が付与されます。

```
False,0
```

の場合は単語番号は使用されません。

---

# historyについて

暗記結果は

```
history/history.json
```

に保存されます。

保存内容は以下の通りです。

- 使用したtable
- 開始時刻
- 継続時間
- 出題モード
- 出題数
- 正答数
- 誤答単語ID

この履歴を利用することで、今後 **苦手単語の出題機能**などを実装する予定です。
